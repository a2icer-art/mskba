# Бизнес‑сценарии

Документ описывает ключевые сценарии, поддерживаемые системой в текущем состоянии.

## Поиск площадок
Пользователь просматривает раздел площадок и фильтрует список по типу. Переходы между типами осуществляются через читаемые URL.

## Управление аккаунтом
Пользователь просматривает данные учетной записи, профиля и разделов, соответствующих ролям участника процесса.

## Управление контактами
Пользователь добавляет контактные данные и подтверждает их через код. Подтвержденные контакты защищены от удаления и редактирования.

## Роли участника процесса
Роли участников отображаются в интерфейсе аккаунта как отдельные разделы, что позволяет расширять функциональность под конкретные роли.

## Каталогизация типов площадок
Администратор настраивает типы площадок и их отображаемые названия. Пользователи видят понятные подписи и URL.

## Верификация контактных данных
Система обеспечивает базовую проверку владения контактом через одноразовый код, предотвращая использование чужих контактных данных.
