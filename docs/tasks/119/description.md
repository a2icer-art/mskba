# task_init_description
Нужно отвязать платежные методы от контрактов: платежные методы хранятся у пользователя, а в контракте выбирается один из методов пользователя. Сохранить привязку методов к площадке. Добавить свойство "метод по умолчанию" для платежных методов, устанавливать автоматически при добавлении первого метода.

# task_short_description
Отвязать методы оплаты от контрактов, выбирать метод пользователя в контракте и добавить флаг метода по умолчанию.
