# 110-4-verify-docs

Цель: зафиксировать проверки и документацию по новой админ‑странице.

## Проверка
- /admin/users доступна только admin (can:admin.access).
- Фильтры работают: поиск, статус, роль, registered_via, даты регистрации.
- Клик по login открывает попап с контактами.
- Сброс подтверждения контакта снимает confirmed_at и удаляет активные верификации.

## Документация
- Обновить описание админ‑разделов и указать новый раздел "Пользователи".
