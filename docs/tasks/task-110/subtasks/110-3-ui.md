# 110-3-ui

Цель: реализовать UI раздела "Пользователи" и попап редактирования.

## Реализация
- Новая страница: resources/js/Pages/Admin/Users.vue.
- Таблица пользователей с кликабельным login.
- Фильтры: поиск, статус, роль, registered_via, диапазон регистрации.
- Попап пользователя:
  - блок системных ролей (чекбоксы) + кнопка сохранения;
  - блок ролей участника (чекбоксы) + кнопка сохранения;
  - список контактов с кнопкой "Сбросить подтверждение".
- Добавлена группа "Пользователи" в admin sidebar.
