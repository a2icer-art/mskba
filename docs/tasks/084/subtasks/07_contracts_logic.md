# 07_contracts_logic

## Ограничения назначения
- Прямое назначение через страницу контрактов доступно только для типа `employee`.
- Типы `owner` и `supervisor` исключены из `getAssignableTypes` и не проходят `assign`.
- Назначение/аннулирование через контрактные права теперь ориентировано на работу с `employee`.

## Контроль
- Проверки оставляют возможность управления контрактами создателем/владельцем/супервайзером (при наличии прав), но без прямого назначения владельца/супервайзера.
