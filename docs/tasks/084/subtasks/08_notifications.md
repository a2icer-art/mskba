# 08_notifications

## Реализация
- Добавлен сервис `ContractNotificationService` для системных сообщений по контрактам и модерации.
- Уведомления отправляются:
  - при подтверждении/отклонении модерации контракта;
  - при назначении контракта;
  - при изменении прав контракта;
  - при аннулировании контракта.
