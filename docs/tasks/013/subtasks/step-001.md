## Текущие роли и точки их использования
Сейчас в системе есть роли доступа: admin, moderator, editor. Это технические роли для администрирования и модерации. Роли участников процесса нужны для бизнес-логики: игры (игрок, тренер, судья), площадки (администратор), медиа/блогер, продавец, персонал. Эти роли не должны смешиваться с ролями доступа.
