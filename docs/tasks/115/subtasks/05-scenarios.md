# Подзадача 05: Сценарии и требования

- Повторная отправка после отклонения.
- Ограничения по времени зависят от настроек площадки.
- Отмена/перенос без возвратов до появления refunded.
- Доступ к доказательствам по правам бронирования.

См. `docs/project/event_organizer_payment_flow.md` (разделы «Ошибки и граничные случаи», «Уведомления», «Ограничения по времени»).
