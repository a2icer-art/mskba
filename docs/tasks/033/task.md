# task_init_description
создать задачу: sidebar меню должно поддерживать сгруппированные ссылки на разделы. пример для Filament: группа "Модерация" с подпунктами "Пользователи", "Площадки", и в будущем другие; группа "Контент" с ссылками по контенту. у каждой группы есть заголовок, как сейчас для всего aside блока.

# task_short_description
Добавить поддержку групп в sidebar-меню (заголовки групп и подпункты), с примером для Filament: "Модерация" и "Контент".

# task_full_description
Нужно расширить компонент бокового меню так, чтобы он мог отображать группы ссылок с заголовками. Требуется сохранить совместимость с текущим форматом плоского списка, чтобы существующие страницы (account/venues) продолжили работать без изменений. На примере Filament меню должно формироваться как набор групп: «Модерация» (с подпунктами, сейчас минимум «Пользователи») и «Контент» (пока пустая, для будущих разделов). Группы без ссылок не отображать. Стили и поведение должны оставаться единообразными с текущим sidebar.

# План выполнения задачи
1. Подготовить структуру групп навигации в сервисе Filament и обеспечить корректный выбор default href. (docs/tasks/033/subtasks/step-001.md)
2. Обновить MainSidebar для поддержки группированных и плоских данных и применить в Filament страницах без поломки текущих разделов. (docs/tasks/033/subtasks/step-002.md)
