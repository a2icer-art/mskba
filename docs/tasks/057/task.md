# task_full_description
Нужно добавить ссылку "Бронирование" в sidebar площадки для пользователей, у которых есть права подтверждения и/или отмены бронирования. Ссылка должна отображаться в группе "Администрация", рядом с пунктом "Контракты". Путь и условия отображения должны соответствовать существующим правилам навигации площадки. Дополнительно требуется расширить задачу на реализацию флоу подтверждения и отмены бронирований: добавить страницу/раздел для просмотра заявок, реализовать действия "Подтвердить" и "Отменить", а также проверки прав и статусов.

# План выполнения задачи
1. Определить точку формирования sidebar площадки и условия для отображения пункта (права подтверждения/отмены). (subtasks/01_nav.md)
2. Добавить пункт "Бронирование" в группу "Администрация" рядом с "Контракты". (subtasks/02_ui.md)
3. Реализовать страницу бронирований площадки: список заявок, фильтрация по статусу. (subtasks/03_bookings_list.md)
4. Реализовать действия подтверждения/отмены с проверкой прав и статусов. (subtasks/04_bookings_actions.md)
5. Обновить документацию и статус задачи. (subtasks/05_docs.md)
