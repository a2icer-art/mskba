# Шаг 2. Frontend: вкладки по ролям участника

## Цель
Отобразить вкладки для ролей участника и базовую информацию о роли.

## План
- Обновить `resources/js/Pages/Account.vue`: добавить вкладки по ролям и секции.
- Сохранить существующие вкладки "Пользователь" и "Профиль".

## Результат
- Для каждой роли участника отображается отдельная вкладка.
