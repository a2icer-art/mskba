# Шаг 1. Backend: роли участника для аккаунта

## Цель
Подготовить список ролей участника пользователя для страницы аккаунта.

## План
- Добавить связь пользователя с назначениями ролей участника.
- В `routes/web.php` сформировать список ролей и передать в Inertia.

## Результат
- Страница аккаунта получает массив ролей участника.
