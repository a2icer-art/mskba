# task_init_description
Необходимо на странице аккаунт помимо вкладок "Пользователь" и "Профиль" добавить вкладки соответствующие связанным ролям-участника у пользователя - например если пользователь при регистрации указал роль игрок - была создана для него PlayerProfile, значит должна появиться и соответствующая вкладка на странице "аккаунт".

# task_short_description
Добавить на странице аккаунта вкладки, соответствующие связанным ролям участника пользователя (например, PlayerProfile).
