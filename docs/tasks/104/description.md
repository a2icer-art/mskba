# task_init_description
Бронирования: в попапе просмотра заявки в поле «Стоимость» по умолчанию показывать стоимость из самого бронирования (не из текущих настроек площадки). Если стоимость бронирования отличается от настройки площадки — добавить «(!)» к label и title «стоимость аренды площадки такая‑то».

# task_short_description
Попап бронирования: дефолтная стоимость из бронирования + индикатор отличия от настроек площадки.
