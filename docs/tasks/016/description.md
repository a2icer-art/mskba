# task_init_description
Создать процесс регистрации пользователя. При котором пользователь обязан: указать логин (уникальный, латинские символы и цифры, знак подчеркивания), пароль (min 6 символов, не простой), email (стандартная валидация). Опционально: роль участника процесса. Добавить кнопку регистрации, в попап авторизации, которая будет переводить на страницу регистрации. По успешной регистрации - необходимо авторизовать пользователя и отправлять на страницу аккаунта.

Сделать регистрацию пользователя с обязательными полями и опциональной ролью.
