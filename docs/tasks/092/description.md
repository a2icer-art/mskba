# task_init_description
На странице /venues/halls/na-dubninskoi/contracts есть секция "Запросы на контракты", в которой пользователи могут подать заявку на контракт типа "Владелец" или "Супервайзер" согласно определенным условиям. Данная секция не должна показываться не должна показываться если эти обе роли уже заняты - то есть если у площадки уже существуют активные контракты с типами "Владелец" и "Супервайзер". Если существует один активный контракт одного типа, например "Владелец" или "Супервайзер" то не показывать в секции плажку того контракта который уже существует а показывать только ту, активного контракта на тип которой еще нет.

# task_short_description
Скрывать секцию запросов или отдельные карточки в ней, если соответствующие контракты уже активны.
