# Шаг 2. Frontend: кнопка подтверждения email

## Цель
Показать кнопку подтверждения рядом с email, если email не подтвержден, и инициировать отправку письма.

## План
- Обновить `resources/js/Pages/Account.vue`: добавить кнопку около email.
- Подключить отправку через Inertia `useForm`.

## Результат
- Кнопка отображается только при неподтвержденном email и отправляет запрос на сервер.
