# Шаг 1. Backend: отправка подтверждения email

## Цель
Добавить эндпоинт для отправки письма подтверждения email и передать в Inertia признак подтверждения.

## План
- Обновить `routes/web.php`: добавить маршрут отправки письма подтверждения.
- В `routes/web.php` расширить props для страницы аккаунта (email_verified_at).

## Результат
- Кнопка "Подтвердить" сможет вызывать серверную отправку письма.
