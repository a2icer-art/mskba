# Подзадача
UseCases/API и валидации управления расписанием.

# UseCases (предложение)
- Просмотр расписания площадки.
- Обновление недельных интервалов.
- Добавление/редактирование исключения на дату.
- Удаление исключения.

# Валидации
- day_of_week: 1..7.
- starts_at/ends_at: валидный time, starts_at < ends_at.
- Отсутствие пересечений интервалов в пределах дня.
- Исключение на дату уникально в рамках schedule.
- Если is_closed = true, интервалы для даты не допускаются.

# Доступ
- Управление расписанием требует ресурсного права venue.schedule.manage.
