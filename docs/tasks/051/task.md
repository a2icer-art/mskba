# task_full_description
Нужно спроектировать сущность VenueSchedule, связанную с Venue (одно расписание на площадку). Расписание влияет на будущие бронирования и учитывает уже занятые временные интервалы. Таймзона фиксированная: UTC+3 (Москва). Требуется поддержка перерывов внутри дня (несколько интервалов доступности). Предусмотреть модель хранения базового расписания по дням недели и механизм исключений/override на конкретные даты. Описать, как расписание взаимодействует с бронированиями (валидация слотов, запрет пересечений).
