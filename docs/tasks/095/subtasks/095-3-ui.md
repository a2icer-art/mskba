# 095-3-ui

Цель: описать UI страницы супервайзера и права доступа (venue.supervisor.view/manage).

- Страница `/venues/{type}/{venue}/admin/supervisor` показывает активного супервайзера и состояние.
- Доступ к разделу — admin или активный супервайзер с `venue.supervisor.view/manage`.
- Редактирование настроек — admin или активный супервайзер с `venue.supervisor.manage`.
- UI: одно поле комиссии + чекбокс «фиксированная надбавка», режим определяет валидируемый диапазон.
