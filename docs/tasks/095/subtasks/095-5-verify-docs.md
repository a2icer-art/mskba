# 095-5-verify-docs

Цель: проверить ключевые сценарии и обновить документацию.

- Проверить отображение страницы супервайзера и сохранение комиссии.
- Убедиться, что комиссия учитывается в сумме бронирования и попапах подтверждения/ожидания оплаты.
- Обновить документацию по текущему состоянию после внедрения.
- Проверить обновление прав супервайзера через /admin/contracts-moderation (для approved заявок).
