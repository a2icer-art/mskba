# 095-1-analysis

Цель: зафиксировать, как супервайзер влияет на стоимость бронирования и связанные процессы (подтверждение, оплата, уведомления).

- Супервайзер — посредник площадки по бронированиям. Комиссия влияет на итоговую стоимость, которая показывается в UI и фиксируется в платеже.
- Комиссия применяется только если у площадки есть активный контракт типа «Супервайзер».
- Итоговая стоимость = базовая стоимость аренды (по настройкам площадки) + комиссия супервайзера.
- Комиссия должна участвовать в расчёте суммы платежа, в карточках бронирования и в попапах подтверждения/ожидания оплаты.
