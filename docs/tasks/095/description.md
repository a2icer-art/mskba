# full_description
Нужно реализовать раздел «Супервайзер» для площадки и добавить логику комиссии, которую управляет супервайзер в рамках бронирований. Супервайзер — посредник, который ведёт площадку по бронированиям, имеет права, связанные только с бронированиями, и не может быть больше одного активного на площадку. Страница должна показывать ключевые параметры супервайзинга (в т.ч. настройку/правила комиссии) и влиять на формирование стоимости бронирования и связанные процессы (подтверждение/оплата/уведомления) согласно архитектуре и процессам проекта.
