# task_init_description
В главном меню ссылку "Залы" (отображение площадок по типу "залы") переделать на "Площадки" (всех типов) и переработать отображение на странице: добавить блок фильтров — горизонтальная полоса сверху компонента, где должны быть минимальные фильтры: select тип, input адрес, предикативный select метро и сортировка select — по названию ASC/DESC, по дате добавления ASC/DESC; также должна быть галочка "группировать по типу", нажатие которой будет отображать площадки, сгруппированные по типу. Далее сам список всех площадок, где площадка должна быть отображена карточкой (как сейчас) во всю ширину и также отображать тип. А внизу — пагинация.

# task_short_description
Переименовать пункт меню в "Площадки" и обновить страницу списка: фильтры, сортировка, группировка, карточки во всю ширину и пагинация.

# full_description
Обновить страницу площадок: заменить пункт меню "Залы" на "Площадки" (все типы), добавить горизонтальный блок фильтров (тип, адрес, метро, сортировка), переключатель группировки по типам, список карточек во всю ширину с отображением типа и блок пагинации.
