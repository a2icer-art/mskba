# task_full_description
Нужно добавить в список площадок `/venues` кнопку создания площадки для подтвержденного пользователя. Кнопка находится справа от заголовка и меняет текст в зависимости от активного подраздела (тип площадки). По нажатию открывается модальное окно с формой. Если пользователь находится в подразделе типа, тип должен быть предвыбран и недоступен для изменения. Обязательные поля формы: название, тип, адрес — как в `VenueModerationRules`. Ошибки выводить в стандартном уведомлении. При успешном создании — редирект на страницу площадки, где отображается статус (аналогично аккаунту: бейдж/кнопки/уведомления) и информация о площадке. Доступ к созданию и отправке на модерацию — только для подтвержденного пользователя.
