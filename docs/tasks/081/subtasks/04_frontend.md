# step-04
Frontend-правки.

# Итог
- Модалка «Принять в работу» расширена полями оплаты.
- Данные отправляются в бекенд в новом формате.
- В списке заявок изменено название действия.

# Изменения
- `resources/js/Pages/VenueBookings.vue`:
  - добавлены пропсы `paymentOrderOptions` и `paymentDefaults`;
  - модалка «Принять в работу»: селект порядка оплаты, срок ожидания оплаты,
    сумма первой части для частичной предоплаты, переключатель минут/часов;
  - логика дефолтов основана на настройках площадки и длительности брони;
  - кнопка в списке переименована в «Принять в работу».

# Поведение
- Для постоплаты скрываются срок ожидания и сумма частичной оплаты.
- Для частичной предоплаты по умолчанию ставится 50% от суммы бронирования.
- Если сумма частичной оплаты достигает полной, автоматически выбирается предоплата.
