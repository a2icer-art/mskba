# task_short_description
Добавить логирование настроек площадки (VenueSettings) в админские логи с корректным отображением в интерфейсе.

## task_details
Нужно включить VenueSettings в аудит (создание/изменение/удаление) и добавить сущность в список фильтров /admin/logs, чтобы логи корректно отображались и фильтровались по настройкам площадки. Использовать текущий механизм AuditLog и единые правила entity-logging.
