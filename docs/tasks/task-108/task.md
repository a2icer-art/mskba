full_description:
Нужно реализовать централизованное управление SEO-метатегами (title/description/keywords) для всех доступных на текущий момент страниц сайта, а также загрузку favicon через админку. 
Редактирование доступно пользователям с ролями admin и editor. 
Редактирование метатегов должно быть построчным в таблице с возможностью сохранить все изменения внизу каждой группы страниц. 
Метатеги связываются с сущностями по идентификатору (id) и типу сущности/страницы. 
Локализация только русская. 
Данные метатегов должны применяться в HTML-мета-тегах соответствующих страниц. 
Favicon загружается в админке и применяется на сайте глобально.

План выполнения задачи:
1) Модель и доступ: миграция/модель page_meta, политика (admin/editor), сервис резолва метаданных.
2) Админ‑интерфейс: раздел SEO с группами страниц, построчное сохранение + «Сохранить все», серверная валидация.
3) Применение метатегов: проброс в props и установка через Head/мета‑теги на страницах.
4) Favicon: загрузка в админке, хранение пути в настройках, подключение в layout.

Результат:
- Добавлены сущности SEO (page_meta) и сервисы резолва/каталога.
- Реализована админ‑страница /admin/seo с построчным и групповым сохранением метатегов и загрузкой favicon.
- Метатеги и favicon применяются глобально через Inertia Head.
- Добавлен глобальный флаг «Включать название сайта в title».
