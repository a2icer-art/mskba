# task_init_description
необходимо воспроизвести механику редактирования площадок аналогично редактированию пользователя. редактировать может только создатель (в будущем — по контрактам/permissions). нужен единый источник определения полей, обязательных для отправки площадки на модерацию и для ограничения редактирования.

# task_short_description
Сделать редактирование площадок (только создатель) и единый источник требований модерации/редактирования.

# task_full_description
Нужно добавить редактирование площадок по аналогии с редактированием пользователя. Редактировать может только создатель площадки. После подтверждения площадки можно редактировать только поля, которые не входят в обязательные для модерации. Требуется создать единый источник (класс/слой) с перечнем обязательных полей площадки для модерации и использовать его и в правилах модерации, и в сервисе редактирования. UI: кнопка «Редактировать» на странице площадки, отдельный попап с формой редактирования.

# План выполнения задачи
1. Создать единый класс требований модерации площадок и подключить к правилам модерации. (docs/tasks/038/subtasks/step-001.md)
2. Реализовать сервис обновления площадки с проверками прав/ограничений и логированием. (docs/tasks/038/subtasks/step-002.md)
3. Добавить маршруты/контроллеры и UI на странице площадки для редактирования. (docs/tasks/038/subtasks/step-003.md)
