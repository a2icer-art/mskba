# Подзадача
Права доступа и роли.

# Права
- venue.booking: создание бронирования через Event.
- venue.booking.confirm: подтверждение заявок на бронирование.
- venue.booking.cancel: отмена заявок на бронирование.

# Роли
- Организатор события: создает Event и booking.
- Администратор/владелец площадки: подтверждает/отклоняет booking.
