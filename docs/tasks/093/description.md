# task_init_description
Правило редактирования прав контрактов:
1) Для всех контрактов, кроме "Владелец": если у текущего контракта есть право "Редактирование прав контрактов", это означает возможность редактировать только те контракты, которые он сам назначил.
2) Исключение для контракта типа "Владелец": при наличии права "Редактирование прав контрактов" он может редактировать права всех чужих контрактов по площадке, включая "создатель", но исключая "супервайзер" (свои контракты редактировать нельзя).

# task_short_description
Уточнить и реализовать правило редактирования прав контрактов: owner может редактировать все чужие (кроме supervisor), остальные — только свои назначения.
