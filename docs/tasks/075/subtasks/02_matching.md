# Сопоставление метро
- сначала пробуем метро из Yandex по названию (нормализация, совпадение по точному/частичному названию)
- если метро не найдено, используем координаты адреса (если есть) и ищем ближайшую станцию в локальном справочнике
- если метро и координаты отсутствуют, сохраняем адрес без метро и оставляем возможность ручной коррекции
