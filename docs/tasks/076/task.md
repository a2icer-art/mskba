# task_init_description
Добавить новый раздел на страницу площадки (например /venues/halls/kapitolii-na-leningradke), по умолчанию активный в sidebar, под названием "О площадке". Это основная публичная посадочная страница с секциями: карта (Yandex) с точкой, расписание на ближайшие две недели с возможностью по клику дня увидеть брони и забронировать (и ссылка "подробнее" на страницу расписания), последние посты (ссылка "подробнее" в будущую "Ленту", раздел пока только в sidebar), отзывы о площадке (ссылка "подробнее" в "Ленту", раздел пока только в sidebar), рейтинг площадки в заголовке h1. Наверху якорные ссылки на секции с плавным скроллом. Секция с картой называется "Адрес": под картой отображать метро с цветным буллетом линии (если установлено), ниже текстовый адрес.

# task_short_description
Сделать публичную страницу "О площадке" с якорями и секциями (карта "Адрес" с метро и адресом, расписание 2 недели с попапом, посты, отзывы, рейтинг) и добавить раздел "Лента" в sidebar как будущий.

# task_full_description
Нужно сделать основную публичную посадочную страницу площадки "О площадке" и включить её в sidebar как активную по умолчанию. Страница содержит якорные ссылки с фиксированным блоком и подсветкой активной секции при скролле. Секции: "Адрес" (интерактивная карта Yandex без контролов, метро с цветным буллетом линии при наличии и текстовый адрес), "Расписание" на 2 недели вперед (формат как на странице расписания, при клике дня — попап с занятыми интервалами и кнопкой бронирования), "Посты" и "Отзывы" (оба ведут в "Ленту" по ссылке "подробнее"), а также рейтинг в заголовке h1 (звезды + число, временно 5 через контроллер). Если метро не задано — строку метро скрывать. Раздел "Лента" в sidebar добавить как будущий с пометкой "в разработке".
