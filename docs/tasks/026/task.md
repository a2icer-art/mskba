# task_init_description
Необходимо на странице "аккаунт" добавить вкладку "Контакты" в которой будет консолидация всех данных имеющих отношение к контактам. На данный момент есть только UserEmail. Во вкладке должны быть fieldset-ы по типам контактов. Для email: возможность добавить, удалить, редактировать (только неподтвержденный), кнопка подтвердить для неподтвержденных, badge "подтвержден" с title = "дата подтверждения". Все действия через ajax с выводом результата в нотификационном блоке под полем. Валидации: у пользователя не может быть двух одинаковых контактов; удалять подтвержденный email нельзя; стандартная валидация email; нельзя добавить новую строку email, если есть несохраненные email.
