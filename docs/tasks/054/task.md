# task_full_description
Нужно реализовать UI для событий и бронирований: страницу списка событий, форму создания события и форму создания бронирования площадки через событие. Событие должно использовать типы EventType (game, training, game_training). Бронирование возможно только через событие и должно учитывать расписание площадки (VenueSchedule) и пересечения бронирований. В интерфейсе предусмотреть базовые статусы, пустые состояния и ограничения по правам (`event.create`, `venue.booking`).
