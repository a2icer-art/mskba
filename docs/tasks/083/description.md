# task_short_description
Проверить автоотмену pending и предупреждение по задаче 082: 1) выставить в настройках площадки pending_review_minutes=120, pending_before_start_minutes=120, pending_warning_minutes=30; 2) создать бронирование со статусом pending на старт через 3 часа; 3) убедиться в предупреждении за 30 минут до cutoff; 4) убедиться в автоотмене и комментарии «Вышло время ожидания подтверждения заявки.».
