# task_init_description
На странице площадок слева используется внутреннее меню для внутренних разделов — применить такой же вариант на странице аккаунт вместо текущих вкладок.

# task_short_description
Заменить вкладки на боковое внутреннее меню на странице аккаунта по аналогии со страницей площадок.

# task_full_description
Нужно заменить табы на странице аккаунта на боковое внутреннее меню слева, визуально и структурно аналогичное внутреннему меню на странице площадок. Контент страниц разделов остается прежним, меняется только способ переключения и верстка. Активный пункт меню должен визуально отличаться, переключение выполнять через Inertia Link с отдельными маршрутами для каждого раздела.
