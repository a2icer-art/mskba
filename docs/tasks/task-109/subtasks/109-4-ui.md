# 109-4-ui

Цель: обновить UI аккаунта для подтверждения Telegram.

## Требования
- Кнопка “Подтвердить” открывает deep-link на бота.
- Отображение статуса ожидания подтверждения.
- Автообновление статуса подтверждения.
- Сообщения об ошибках/истечении токена.

## Реализация
- В Account.vue добавлена обработка flash.telegram_verification (deep-link).
- Для Telegram скрыт ввод кода, показаны подсказки и кнопка «Открыть бота».
- Автообновление статуса: polling contacts/contactVerifications каждые 8 секунд при ожидании подтверждения.
