# task_short_description
UX‑обработка истекшего CSRF (419).

# task_solution_options
1. Авто‑обновление CSRF: перехватывать 419 и автоматически перезагружать страницу или повторять запрос с новым токеном.
2. Мягкое уведомление: показать пользователю сообщение “Сессия устарела, обновите страницу” с кнопкой “Обновить”.
