# task_init_description
При подтверждении заявки на контракт с площадкой типа "Владелец" или "Супервайзер" из соответствующего раздела модерации (/admin/contracts-moderation) необходимо в подтверждающий попап добавить возможность задать период действия контракта (от и до), со значением "от" по умолчанию - сегодня.

# task_short_description
Добавить в попап подтверждения контракта поля периода действия (от/до) с дефолтным значением "от" = сегодня.
