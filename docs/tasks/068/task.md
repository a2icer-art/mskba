# task_short_description
Сущность баланса пользователя с операциями пополнения, холда, списания и блокировки.

# task_full_description
Нужна доменная сущность баланса пользователя, поддерживающая операции пополнения, удержания (hold), списания и блокировки с указанием причины блокировки. Должна быть расширяемой и совместимой с будущими платежами/бронированиями. Также требуется добавить страницу в аккаунте (/account/balance) и админский раздел /admin/balances с управлением балансами пользователей.
