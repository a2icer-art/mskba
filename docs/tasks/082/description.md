# task_short_description
Добавить автоотмену заявок со статусом pending по времени начала, учитывая lead time площадки.

# task_details
- Гибридный сценарий автоотмены pending:
  - pending_review_minutes: автоотмена по времени рассмотрения заявки (с учетом рабочего времени площадки).
  - pending_before_start_minutes: автоотмена по приближению к старту (жесткий cutoff).
- Отдельная настройка pending_warning_minutes:
  - уведомлять за N минут до автоотмены;
  - если значение 0 — уведомления не отправлять.
- Рабочее время: использовать расписание площадки (VenueSchedule + исключения).
