# UI-сценарии проверки
1. В настройках площадки задать:
   - `pending_review_minutes = 120`,
   - `pending_before_start_minutes = 120`,
   - `pending_warning_minutes = 30`.
2. Создать бронирование со статусом `pending` и временем старта через 3 часа.
3. Убедиться, что за 30 минут до cutoff приходит предупреждение.
4. Убедиться, что по достижению условия заявка переводится в `cancelled`
   с комментарием «Вышло время ожидания подтверждения заявки.»
