# step-01
Аудит текущих механизмов.

# Найдено
- Автоотмена есть только для `awaiting_payment` через `BookingPaymentExpiryService`.
  Основание — `payment_due_at`. Комментарий при отмене: «Вышел срок на оплату.»
- Запуск: `RunBookingPaymentExpiryJob` по расписанию в `routes/console.php`
  (каждую минуту) + ручная команда `bookings:expire`.
- Дополнительно в контроллере `VenuesController` есть точечные проверки
  `cancelIfExpired` при действиях с оплатой.

# Отсутствует
- Нет автоотмены для `pending`.
- Нет предупреждений за N минут до автоотмены.
- Нет учета рабочего времени площадки для таймеров.
