# step-02
Правила и данные.

# Правила
- Механизмы автоотмены разделены по статусам:
  - `pending`: работают гибридные правила (review + cutoff).
  - `awaiting_payment`: работает только `payment_due_at`.
- После перевода в `awaiting_payment` правила pending больше не применяются.

# Новые параметры настроек площадки
- `pending_review_minutes` — время на рассмотрение заявки (учет рабочего времени площадки).
- `pending_before_start_minutes` — cutoff до начала события (без учета рабочего времени).
- `pending_warning_minutes` — предупреждение за N минут до автоотмены (0 = не уведомлять).

# Комментарии и уведомления
- При автоотмене по pending: «Вышло время ожидания подтверждения заявки.»
- При уведомлении: «Заявка будет автоматически отменена через N минут, если не будет подтверждена.»
