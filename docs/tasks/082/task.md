# task_full_description
Нужно добавить гибридную автоотмену заявок со статусом `pending`.
Первая ветка — по времени рассмотрения заявки: параметр `pending_review_minutes` (минуты, учитываем
только рабочее время площадки по расписанию и исключениям).
Вторая ветка — по приближению к старту: параметр `pending_before_start_minutes` (жесткий cutoff
по времени начала бронирования, независимо от рабочего времени).

Добавить настройку `pending_warning_minutes` для уведомления о скорой автоотмене.
Если значение равно 0 — уведомления не отправляем.

Нужны изменения в настройках площадки, в воркере/сервисе актуализации бронирований,
а также системные уведомления и комментарии при автоотмене.
