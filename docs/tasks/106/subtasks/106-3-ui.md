# 106-3-ui

Цель: отразить правила ролей/лимитов в UI (фильтр ролей, счетчики).

## Выполнено
- UI получает allowed_roles и limit_role из EventsController.
- Списки ролей в формах участия/приглашения фильтруются по allowed_roles.
- Счетчики участников/резерва считают только limit_role (player).
- Публичный интерфейс учитывает лимит только для limit_role при самозаписи и подтверждении.
